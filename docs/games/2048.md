# 2048 gameplay guide

## Overview

2048 is a sliding tile puzzle where each move shifts all tiles in one direction and merges matching values.

## Objective

Reach the 2048 tile while maximizing your score.

## Controls

| Input | Action |
|------|--------|
| Arrow keys / WASD | Move tiles |
| Swipe | Move tiles |
| R | Restart current run |
| Click/tap/Enter/Space on title or game over | Start or restart |

## Gameplay rules

- Every valid move slides tiles and then spawns one new tile (2 most of the time, sometimes 4).
- Tiles merge only once per move.
- Score increases by the value of merged tiles.
- The game ends when the board is full and no merges remain.
